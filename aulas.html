  • Listas

    • Conceito

    Em ciência da computação, uma lista (ou sequência) é uma estrutura de dados abstrata que
    implementa uma coleção ordenada de valores, onde o mesmo valor pode ocorrer mais de uma vez.
    Uma lista é uma sequência finita de elementos.
    Cada elemento da lista é identificado por um índice que indica sua posição na sequência.

    Funcionamento:

    - Armazenamento: A lista armazena os dados de forma sequencial.
    - Elementos: Os elementos são organizados em uma ordem específica.
    - Índices: Cada elemento é acessado através de um índice.
    - Repetição: Um mesmo valor pode ocorrer **mais de uma vez na lista.
    Existem diferentes tipos de listas, como listas ligadas e listas duplamente ligadas.
    Nas listas ligadas, os nós são amarrados indicando qual é o próximo nó.
    Já nas listas duplamente ligadas, os nós conhecem tanto o próximo quanto o nó anterior.

  • Árvore Binária

    • Conceito

    Uma árvore binária é uma estrutura de dados hierárquica que representa relações entre elementos.
    Uma árvore binária é composta por nós (ou vértices) e arestas (ou ramificações) que conectam os nós.
    Cada nó pode ter no máximo dois filhos: um filho esquerdo e um filho direito.

    Hierarquia:

    A organização dos nós em uma árvore binária é hierárquica.
    O primeiro nó é chamado de raiz.
    Se a raiz tem filhos, ela é um nó pai, e os filhos são nós filhos.
    Cada nó contém um valor ou dados.
    Os nós podem ter zero, um ou dois filhos.
    As ligações entre os nós são as arestas.

    Exemplos:

    Imagine uma árvore genealógica com avós, pais e filhos. Ela é uma estrutura hierárquica.

  • Grafos

    • Conceito

    Um grafo é uma estrutura de dados que representa um conjunto de objetos chamados vértices, interligados por meio de conexões chamadas arestas.
    Um grafo pode ser definido como um par (V, A), onde:
    V: conjunto de nós (vértices).
    A: conjunto de pares de vértices (arestas).
    Tanto vértices quanto arestas podem armazenar elementos.
    Quando as arestas armazenam grandezas numéricas, o grafo é dito ponderado.

    Terminologia:

    Vértice final de uma aresta: Os vértices finais (ou endpoints) de uma aresta são chamados de U e V.
    Arestas incidentes em um vértice: As arestas a e b são incidentes no vértice V.
    Vértices adjacentes: Os vértices U e V são adjacentes.
    Grau de um vértice: O vértice X tem grau 5 (número de arestas incidentes em X).
    Laços: Um vértice que se conecta a si mesmo é chamado de laço.
    Arestas paralelas: A e B são arestas paralelas se compartilham vértices finais em comum.
    Subgrafo: Um subgrafo de um grafo G é um grafo que contém um subconjunto dos vértices e arestas de G.
    Caminho: Uma sequência alternante de vértices e arestas que começa e termina em vértices.
    Ciclo: Um caminho circular (o primeiro e último vértices são iguais).
